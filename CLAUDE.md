# El Radar del Airsoft

Directorio web de la comunidad de airsoft en Perú (canchas, tiendas, grupos de WhatsApp,
eventos, importadores, mercado de pulgas, workshops, jugadores, equipos, marco legal).
Sin backend, sin login — todo el contenido vive en `public/data/airsoft.json`.

Ver [README.md](README.md) para stack, estructura de carpetas y cómo editar el contenido.

## Design Context

Este proyecto usa el skill `impeccable` para trabajo de diseño. Antes de tocar UI, lee:

- [PRODUCT.md](PRODUCT.md) — register (`product`), usuarios, propósito, posicionamiento,
  personalidad de marca, anti-referencias, principios de diseño.
- `DESIGN.md` — aún no generado. Correr `/impeccable document` cuando se necesite capturar
  el sistema visual actual (paleta, tipografía, componentes) en un documento formal.

Resumen rápido: directorio funcional ("product", no "brand") para jugadores de airsoft en
Perú que buscan dónde jugar/comprar/unirse a un grupo. Estética táctica pero cercana —
negro + verde oliva (`accent: #6B9B37`), Oswald en mayúsculas trackeadas para display,
esquinas rectas — sin caer en look "neon gamer" ni en plantilla genérica de directorio.
